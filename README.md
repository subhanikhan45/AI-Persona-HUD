# 🤖 AI Persona HUD (Neural HUD v1.0)

A real-time, high-tech AI dashboard that detects facial emotions and assigns a "Digital Persona" using Computer Vision. Built in 48 hours for the HP EliteBook 840 G5.

## 🚀 Features
* **Live Rapid Detection:** Uses DeepFace & OpenCV for sub-100ms emotion tracking.
* **Persona Mapping:** Dynamically switches between *The Architect*, *The Warrior*, and *The Optimist*.
* **Cyberpunk UI:** Includes scanline filters, blinking REC indicators, and neon-themed bounding boxes.
* **Audio Feedback:** Integrated `winsound` triggers for persona transitions.

## 🛠️ Tech Stack
* **Language:** Python 3.12
* **Libraries:** OpenCV, DeepFace, TensorFlow
* **Hardware:** Optimized for Intel UHD 620 Graphics (Integrated)
